Introduction

The galaxy simulator uses current models of the components of galaxies in order to accurately model and simulate them, employing advanced probability distributions to understand and generate stars. The program returns a csv file with (x,y,z) coordinates as well as the brightness, temperature, and size for individual objects, that can be run by OzView or ParaView to render in 3D space.
